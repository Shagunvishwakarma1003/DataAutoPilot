# 🚀 DataAutoPilot — Automated Data Science Pipeline

An end-to-end *AutoML-style pipeline* that automatically performs:

* Exploratory Data Analysis (EDA)  
* Automatic target detection  
* Machine Learning model training  
* Model leaderboard comparison  
* Feature importance analysis  
* SHAP explainability  

Users can simply upload a dataset and the system will automatically build the ML pipeline.
